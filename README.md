# 👻絵文字配布物について

このリポジトリで配布している絵文字の利用規約です。

## ライセンス

### 許可されていること

* 個人利用の範囲で、自由に保存・利用していただいて構いません。

### 禁止事項

以下の行為を固く禁じます。

* **自作発言**: 作者であると偽る、または示唆する行為。
* **商用利用**: 利益を得る目的での利用。
* **再配布**:
    * 原則として、データの再配布は禁止です。
    * **例外**: Fediverse（Misskey, Mastodon, Pleromaなど）のサーバーへ絵文字としてインポート（輸入）する形での利用は許可します。
